# Shell编程




- 使脚本具有执行权限：`chmod +x ./test.sh`
- 变量名和等号之间不能有空格
- 只读变量：
```
a=???
readonly a
```
- 多行注释
```
:<<EOF
内容
内容
EOF
(EOF为一种符号，比如!,"!@#$%^)
```
- 条件表达式要放在方括号之间，并且要有空格，例如: [$a==$b] 是错误的，必须写成 [ $a == $b ]
- -eq：相等返回true
- -ne：不相等返回true
- -gt：左边大于右边，返回true
- -lt：左边小于右边，返回true
- -ge：左边大于等于右边，返回true
- -le：左边小于等于右边，返回true
- -o：布尔运算符或运算
- -a：布尔运算符与运算
- 生成随机数：%s%N，%s是秒数，%N是当前时间的纳秒数据，精确到亿分之一秒
